Sujet d'Évaluation : Création de Site Web
Durée : 3 heures
Objectif : Créer un site avec un thème au choix (ex: Portfolio, Site
vitrine, etc.)
1. Structure Obligatoire (Base du site)
Le site doit respecter l'ordre et les éléments suivants (de haut en
bas):
1. Navbar (Barre de navigation) :
o Un Logo.
o 3 ou 4 liens de navigation.
2. Hero Section (Bannière) :
o Image de fond ou d'illustration.
o Un Titre principal.
o Un Bouton (Call to Action).
3. Section "Cards" (ex: Services, Avantages) :
o 3 cartes alignées horizontalement.
o Note technique : Utiliser Flexbox ou CSS Grid.
4. Section Présentation :
o Mise en page "Split" (divisée en deux) : 50% Image / 50%
Texte.
5. Formulaire Basique :
o Champ Email.
o Champ Message.
o Bouton d'envoi.
6. Footer (Pied de page) basique :
o Mention du nom / Copyright.

2. Barème de Notation (Total /20)
A. Les Critères de base (Note sur 15/20)
• Structure (5 pts) : Respect des éléments obligatoires listés cidessus (sans ce critère validé , impossible de valider le reste)
• Responsive (5 pts) : Le site doit s'adapter correctement sur
Mobile et Ordinateur.
• Performance (5 pts) :
o Images optimisées.
o Score Lighthouse : 90/100.
B. Les Bonus (Pour passer de 15/20 à 20/20)
Conditions : Ces points ne sont comptabilisés que si vous avez
obtenu 15/20.
• Hébergement (+2 pts) : Mettre le site en ligne via GitHub
Pages.
• Sémantique & Perf (+1 pt) : Utilisation du Lazy Loading et des
balises sémantiques HTML5 correctes (<header>, <section>,
<footer>, <nav>).
• Fonctionnalité (+1 pt) : Création d'une Section FAQ dépliable
(accordéon).
• Mobile (+1 pt) : Mise en place d'un Menu Burger pour la
version mobile.
